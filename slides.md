# How to Initialize a New Repository
```console
git init
```
---
# How to review changes
Start by checking the status of your files:
```console
git status
```
Then use...
```console
git diff
```
---
# How to commit changes

```console
git add README.md
git commit -m "Initial commit"
```
---
# Create a new branch

```console
git checkout -b feature/branching
```
